# 安装步骤

1. 克隆仓库：
    ```sh
    git clone <repository-url>
    ```
2. 导航到项目目录：
    ```sh
    cd tool/home-fin/code
    ```
3. 安装依赖：
    ```sh
    npm install
    ```

## 使用方法

1. 启动项目：
    ```sh
    npm start
    ```
2. 打开浏览器并访问 `index.html` 文件，输入相关信息并点击“计算”按钮查看结果。
